    Classifier Learning Center

    Explore the concepts and math supporting text preprocessing and machine learning text classification.

    Before text can be classified by machine learning models, it must be normalized and converted into numerical vectors. This follows a structured natural language processing pipeline:

    • Tokenization: Breaking the raw text string into individual words while stripping special characters and converting to lowercase.
    • Stopword Filtering: Removing extremely common words (e.g., 'and', 'the', 'of') that carry little conceptual context. This reduces vocabulary size.
    • Stemming: Reducing words to their base roots (e.g., 'arguing', 'argues', 'argued' all simplify to 'argu'). This groups variations of a word together.
    // NLP Preprocessor Pipeline Example Input: "SHOCKING: Doctors Can't Believe..." 1. Tokenize -> ["shocking", "doctors", "can't", "believe"] 2. Clean -> ["shocking", "doctors", "believe"] (remove stopwords) 3. Stem -> ["shock", "doctor", "believ"]

    TF-IDF measures how important a term is to a document relative to a corpus of documents. It combines two core metrics:

    • Term Frequency (TF): The relative rate of the term in the document. Calculated as count of word divided by total words in that document.
    • Inverse Document Frequency (IDF): Measures how rare a word is in the entire training corpus. Rare words get higher weight; common words like "said" get downweighted.
    • TF-IDF Weight: Found by multiplying TF by IDF. The resulting vector represents the document's numerical footprint.
    TF(t, d) = Count(t in d) / Total Words in d IDF(t) = log(1 + (Total Docs / (1 + Docs Containing t))) Weight(t, d) = TF(t, d) * IDF(t)

    Naive Bayes uses Bayes' Theorem to predict classification labels based on the conditional probabilities of tokens. The "naive" assumption is that words appear independently of each other given the class.

    • Prior Probability: The general probability of seeing a class before reading any words (e.g. 50% Real / 50% Fake).
    • Word Likelihood: The probability of a word appearing in articles of a specific class: $P(\text{Word} \mid \text{Class})$.
    • Laplace Smoothing: Adding 1 to all word counts so that words unseen in training don't force a zero probability result.
    Bayes Prediction Formula: P(Class | Document) ∝ P(Class) * ∏ P(Word_i | Class) Laplace Word Likelihood: P(Word | Class) = (Count(Word in Class) + 1) / (Total Words in Class + Vocab Size)

    Logistic Regression computes a weighted linear combination of TF-IDF feature weights, and squashes the outcome between 0 and 1 using the Sigmoid activation function to denote real/fake probability.

    • Weights (coefficients): Each word has a weight. Positively associated words pull the prediction toward "Real News" (label 1), while clickbait/sensationalist words have negative weights pulling it toward "Fake News" (label 0).
    • SGD Optimization: The model starts with zero weights and iterates through the training set, adjusting weights step-by-step to minimize classification errors.
    Activation: P(y=1|x) = Sigmoid(∑ w_i * x_i + bias) Sigmoid(z) = 1 / (1 + e^-z) SGD Weight Update Rule: w_i ← w_i - alpha * ((Prediction - Target) * x_i + lambda * w_i)

    The Passive Aggressive Classifier is an online learning algorithm ideal for text classification. It remains passive if the prediction is correct and confident, but aggressive if incorrect, updating weights directly to correct the error.

    • Passive Phase: If a sample prediction matches the label and has a margin score of at least 1, the classifier updates nothing (weights remain identical).
    • Aggressive Phase: If there's an error or narrow margin, weights are updated by a step size $\tau$ to correct the model instantly.
    • Regularization: Aggressiveness parameter $C$ keeps weight adjustments bounded, preventing outliers from destabilizing the model.
    Hinge Loss: L = max(0, 1 - Target_y * WeightDotInput) Update Step: tau = Loss / (||InputVector||^2 + 1/(2*C)) Weight Update Rule: w ← w + tau * Target_y * InputVector
